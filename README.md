# Library Management System 🏛️

A Python project built to demonstrate all 6 core OOP concepts.

## Concepts Covered
- Classes & Objects
- Inheritance
- Encapsulation
- Polymorphism
- Abstraction
- Object Lists

## Features
- Add books to library
- Issue & return books
- Member borrow limits (Student: 2, Teacher: 5)
- Track borrowed books per member

## Built With
- Python (Pydroid 3 on Android)
